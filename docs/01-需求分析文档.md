# 需求分析文档

> 高校实验室设备借用管理系统  
> 版本：v1.0 | 日期：2026-06-16

---

## 1. 项目背景

高校实验室设备种类多、借用频繁，传统纸质登记方式效率低、易出错、难追溯。本系统实现设备借用的在线化管理，提升实验室管理效率。

## 2. 用户角色

| 角色 | 权限说明 |
|------|----------|
| **管理员** | 设备管理、借用审批、用户管理、数据统计 |
| **普通用户**（学生/教师） | 浏览设备、提交借用申请、查看借用记录 |

## 3. 功能需求

### 3.1 设备管理
- [ ] 添加设备（名称、编号、型号、状态）
- [ ] 编辑设备信息
- [ ] 删除设备
- [ ] 查询设备（按名称/编号/状态筛选）

### 3.2 借用管理
- [ ] 提交借用申请
- [ ] 审批借用申请（通过/拒绝）
- [ ] 归还设备
- [ ] 查看借用记录

### 3.3 用户管理
- [ ] 用户注册
- [ ] 用户登录
- [ ] 管理员管理用户

## 5. 非功能需求

| 需求类型 | 要求 |
|----------|------|
| 性能 | 页面响应 < 2秒 |
| 可用性 | 界面简洁，操作步骤 ≤ 3步 |
| 安全性 | 密码加密存储，角色权限控制 |
| 兼容性 | 支持 Chrome/Firefox/Edge 浏览器 |

## 6. 运行环境

- **后端**: Python 3.x + Flask
- **数据库**: SQLite
- **前端**: HTML + CSS + JavaScript（单页应用）
- **浏览器**: Chrome 90+, Firefox 88+, Edge 90+
